# Исторические даты

Интерактивное приложение для просмотра исторических событий по годам.

## Технологический стек

- React
- TypeScript
- Styled Components
- Swiper

## Установка и запуск

1. Клонируйте репозиторий:

```bash
git clone [ссылка на репозиторий]
```

2. Перейдите в директорию проекта:

```bash
cd history-dates
```

3. Установите зависимости:

```bash
npm install
```

2. Запустите проект:

```bash
npm run dev
```
